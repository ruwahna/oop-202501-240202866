### 7. Lessons Learned

1. **OOP Principles**: Penerapan SOLID principles meningkatkan maintainability
2. **Design Patterns**: Strategy pattern untuk payment methods sangat extensible
3. **Layered Architecture**: Pemisahan concerns membuat testing lebih mudah
4. **Team Collaboration**: Communication is key dalam team project
5. **Documentation**: Documentation membantu understanding dan handover
